# Deep Learning Lab

A comprehensive deep learning laboratory containing multiple notebooks covering fundamental and advanced concepts in neural networks and deep learning.

## Overview

This repository contains practical implementations and experiments in deep learning, organized as a series of Jupyter notebooks with detailed explanations, code examples, and visualizations.

## Contents

The repository includes the following Jupyter notebooks:

- **lab 1.ipynb** - Introduction to deep learning fundamentals including neural networks basics, activation functions, and foundational concepts
- **lab 2.ipynb** - Advanced deep learning topics covering optimization techniques, advanced architectures, and practical applications

## Features

The labs cover essential deep learning topics including but not limited to:

- Neural network fundamentals and architecture
- Activation functions and their properties
- Forward and backpropagation algorithms
- Optimization techniques (SGD, Adam, RMSprop, etc.)
- Loss functions and their applications
- Model training, validation, and evaluation
- Deep learning architectures (CNNs, RNNs, etc.)
- Practical deep learning applications

## Requirements

To run these notebooks, you'll need:

- Python 3.7+
- Jupyter Notebook or JupyterLab
- NumPy
- Pandas
- Matplotlib
- TensorFlow or PyTorch (depending on implementations)
- Scikit-learn

Install dependencies with:

```bash
pip install jupyter numpy pandas matplotlib tensorflow scikit-learn
```

## Usage

1. Clone this repository
2. Install the required dependencies
3. Launch Jupyter Notebook or JupyterLab
4. Open any of the .ipynb files to explore the experiments

## Getting Started

**Beginners:** Start with **lab 1.ipynb** for a comprehensive introduction to deep learning fundamentals and core concepts.

**Advanced learners:** Progress to **lab 2.ipynb** for advanced topics and practical implementations.

## How to Run

```bash
jupyter notebook
```

Then select the notebook you want to explore from the Jupyter interface.
